How to install

1
Extract the file
Open your terminal and run:
tar -xzf kdp-novel-factory-v1.0.0.tar.gz
Or double-click the file on Mac/Windows to extract.
2
Install via OpenClaw
In your OpenClaw dashboard, go to Skills → Install from folder and select the extracted kdp-novel-factory folder. Or copy the folder into your OpenClaw skills directory.
3
Start a conversation
Open a chat with your Claw. It will detect the new skill and walk you through the setup — pen name, Replicate API key, genre preference, and communication channel. Takes about 5 minutes.
4
Tell it what to publish
Once setup is complete, say: "I want to write a [genre] novel about [idea]." The factory takes it from there.

Questions? Ask your Claw first — it knows the system inside out.
